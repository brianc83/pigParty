# pigParty

> https://brianc83.github.io/pigParty/

### Workflow

1. Use LEOCad => Save as `.mpd`
2. Move file into `ldraw/models`
3. Run `node packLDrawModel.mjs models/<x>.mpd`
4. Move `models/<x>mpd_Packed.mpd` into project models and import

### Run locally
```
npm run dev
```
> http://localhost:5173
